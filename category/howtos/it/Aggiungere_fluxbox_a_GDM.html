Agguingere fluxbox a GDM

1. Introduzione

Quando viene installato fluxbox da sorgenti, nel caso in cui utilizziate un Desktop Manager come GDM, e' necessario aggiungere una voce nel menu di GDM che permetta di selezionare fluxbox al login.

Per fare cio' e' necessario creare un file chiamato fluxbox.desktop nella directory /usr/share/xsessions/ e inserire al suo interno delle istruzioni che variano a seconda del tipo di distribuzione utilizzata.

1.1 Debian

	[Desktop Entry]
	Name=Fluxbox
	Comment=Highly configurable and low resource X11 Window manager
	Exec=/usr/local/bin/startfluxbox
	Terminal=false
	TryExec=/usr/local/bin/startfluxbox
	Type=Application

	[X-Window Manager]
	SessionManaged=true

Nota: /usr/local/bin/startfluxbox e' il percorso predefinito usato compilando da sorgenti.

1.2 Slackware

	[DesktopEntry]
	Encoding=UTF-8
	Type=XSession
	Name=Fluxbox
	Exec=/etc/X11/xinit/xinitrc.fluxbox

Salvate il file e cambiate i suoi permessi attraverso il seguente comando:

chmoda+x/usr/share/xsession/fluxbox.desktop

Al riavvio di GDM nel menu che vi permette di scegliere il window manager dovrebbe apparire una voce in piu' relativa a fluxbox.
