howto 창 테두리만 보이기

플럭스박스를 실행하고 있지 않을 때 ~/.fluxbox/apps를 수정하라. 따라서 플럭스박스를 종료하고 파일을 열어라. 아래 내용을 추가할 것:

  [app] (.*)
     [Deco]    {BORDER}
  [end]

저장하고선 플럭스박스를 다시 시작하라. 모든 유틸들이 이제 오직 border만 보일 것이다.(스타일이 border를 설정했다면). 사용자가 border(테두리)의 픽셀 너비와 색깔을 바꿀 수 있다.

overlay 파일을 사용하여 모든 스타일에 포괄적으로 적용하거나 스타일마다 각각 따로 적용한다.

두 경우 모두 해당 파일에 두 줄을 덧붙여야 한다:

  window.borderWidth:   
  window.borderColor:   color (e.g. #009900, black, rgb:00/64/00)

의미하는 바를 쉽게 알 것이다, int값은 border의 너비 픽셀값을 정하고, color는 border의 실제 색깔이다.

확실하게 하기 위해, 스타일 파일에 이것을 첨가할 때는, 가장 아랫줄에 저 두 줄을 적어라. 이렇게 하면 어떤 것도 주석 처리할 필요 없이 스타일 파일의 원본을 유지하고선(원본으로 나중에 다시 쓸 지 모른다), 원하는 설정값을 사용할 수 있다.

howto 창 테두리와 탭만 보이기

외부탭이 플럭스박스 1.0rc에 돌아온 이후 일반적인 방법(탭이 온전한 창테두리 위에 놓이기) 혹은 특별한 방법으로 그것을 이용할 수 있다. 위의 사용자 ~/.fluxbox/apps 파일에 아래의 지침 내용처럼 적어라:

  [app] (.*)
     [Deco]    {TAB}
  [end] 

창에 테두리만 보이지만(위에서처럼) 루트메뉴(rootmenu)의 tab 섹션에서 설정한 대로 멋진 탭이 추가로 표시될 것이다.
