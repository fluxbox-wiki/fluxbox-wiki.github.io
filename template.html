0. Introduction

The aim of this page is to provide a template to easily create all the pages of the website fluxbox-wiki.github.io with a consistent style. The code of this page contains all the structures that may be needed in order to write all the pages. Should that template won't be enough for your needs just post on the github project and we will endeavor to overcome every limit. For further help please refer to the code of this page that contains all the structures and all the comments that you may need.
